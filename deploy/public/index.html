Storefunctions

Aplicación de gestión de inventario

Windows (.zip) Android (.apk)

Windows

  1. Descarga el archivo .zip
  2. Extrae en cualquier carpeta
  3. Ejecuta storefunctions.exe
  4. No requiere instalación

Android (Tablet)

  1. Descarga el archivo .apk
  2. Abre el archivo en tu tablet
  3. Permite la instalación de fuentes desconocidas
  4. Instala y abre la app
